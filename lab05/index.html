Lab03 Summary

In lab 03, It was more of an introduction to CSS, which stands for Cascasding Style Sheets. Css is the language that is used to describe how to display HTML file. It defines the type of color, text, background prperties, and more. Css has no syntax compare to html, however, css uses selectors to impliment properties. The rule of a css are comprised of selector, prperty, statement and values.HTML selectors define how element should be displayed in between the opening and closing of an html tags. An ID selector is used to apply specific properties to only one element. Lastly, I learned also about the Class Selector which can be used to define properties to any number of elements via the class attribute.
